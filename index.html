Native Web APIs Repository

This repository contains examples and demos of various Native Web APIs, organized into separate folders based on their respective APIs. Each folder contains practical examples and explanations to help you understand and utilize these powerful web technologies.

Table of Contents

  • History API: Manipulate the browser's history and navigate between pages without full page reloads.
  • Intersection Observer API: Detect when an element enters or exits the viewport.
  • requestAnimationFrame API: Efficiently animate elements using the browser's native animation capabilities.
  • Clipboard API: Access and modify the clipboard contents programmatically.
  • Battery API: Retrieve battery status and monitor changes in battery level and charging status.
  • Date API: Work with dates and times in JavaScript.
  • FormData API: Create and manipulate `FormData` objects for easy form submissions.
  • Fetch API: Perform network requests using the modern Fetch API.
  • Media Capture API: Access and manipulate media devices like cameras and microphones.
  • Contact Picker API: Pick contacts from the user's address book.
  • Vibration API: Control the device's vibration capabilities.
  • Geolocation API: Retrieve the user's geographical position.
  • Notification API: Create native notifications to alert users.
  • File API: Work with files on the user's system.
  • Device Orientation API: Retrieve device orientation and motion data.
  • Canvas API: Draw and manipulate graphics using the HTML canvas element.
  • Web Animation API: Create complex animations using the Web Animation API.
  • IndexedDB API: Store large amounts of structured data on the client-side.
  • Permission API: Check and request user permissions for various actions.
  • Local Storage: Use local storage to store key-value pairs persistently.
  • Service Worker API: Implement service workers for offline capabilities and background sync.
  • Web Workers: Run JavaScript in the background using web workers for improved performance.
  • URL API: Parse and manipulate URLs easily.
  • Push API: Implement push notifications for your web app.
  • UI Events API: Explore various UI events and their applications.
  • Web Audio API: Manipulate audio in the browser, create music, or process audio signals.
  • Page Visibility API: Determine the visibility state of the page.
  • Dataset API: Work with custom data attributes on HTML elements.
  • Payment Request API: Request payments securely from users.

How to Use

Each folder will contain its own README file with detailed explanations and examples for the respective API. Navigate to the specific folder you are interested in and follow the instructions provided in the README file.
